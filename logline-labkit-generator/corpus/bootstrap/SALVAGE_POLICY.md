# Salvage Policy

`Archive(1).zip` is not the base repository.
It may be used only as salvage input.

Any reused material must be listed in `SALVAGE_REPORT.md` with:

- original path
- new path
- reason reused
- modifications applied
- authority status
- whether human review is required

Do not inherit old structure by default.
Do not preserve contaminated authority files.
Do not use old README files as authority.
Do not reuse edited vendor canon.
Do not import docs/decisions or docs/inventory as product authority.
