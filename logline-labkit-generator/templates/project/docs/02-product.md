# Product

The kit provides an installable local product surface plus command-line operation, manifests, schemas, examples, and honest reports for partial or ghosted surfaces.

The first screen is `logline-lab serve`: a local browser operator that creates the first Lab home, runs the setup cycle, displays Load / Declare / Observe / Emit / Project / Learn, and keeps raw command output visible for support and audit.

The browser product remains local workspace state only. It does not claim official spine, receipt, evidence proof, remote sync, or LLM authority.
