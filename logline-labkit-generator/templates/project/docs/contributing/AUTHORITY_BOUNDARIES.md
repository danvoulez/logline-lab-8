# Authority Boundaries

This repository keeps authority roles separate so PRs cannot silently upgrade implementation material into canon.

- Canon loads.
- Pack interprets.
- Profile provides capabilities.
- Lab runs.
- Projection reads.
- Runtime observes/executes.
- LLM suggests Candidates only.

Dan-approved corpus is planning/source input, not implementation receipt.

Foundation canon is not amended by this repository unless explicitly approved by Dan through a separate canon/governance process.

Vendor material is immutable.
